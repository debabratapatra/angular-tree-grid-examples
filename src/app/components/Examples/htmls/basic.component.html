Basic Tree Grid

This is a very basic grid. You provide an array and a configuration.

Data

    {{"{id: 1, name: 'Bimal', age: 60, weight: 60, gender: 1, phone: 7930343463, parent: 0}"}}
    {{"{id: 2, name: 'Bhagi', age: 40, weight: 95, gender: 1, phone: 7930343463, parent: 1}"}}

Configuration

id_field and parent_id_field are mandatory. 
    id_field: 'id'
    parent_id_field: 'parent'

Column

name is mandatory. It is the key in Data. 
    name: 'name'
    header: 'Name'
    width: '100px'

For customized value, renderer function can be used(See below). Or you can use Custom View Component altogether.

Renderer

    {{"renderer: function(value) {"}}
    {{"    return value + ' years';"}}
    {{"}"}}
