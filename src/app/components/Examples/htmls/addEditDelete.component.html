With Add, Edit and Delete Option

Icon class for Expand, Collapse, Edit, Cancel, Delete and Add can be given under css property of the Grid.

CSS

Icons can be customized by setting a css class. Please check below. 
    {{"css: { "}}
    {{"    expand_class: 'fa fa-caret-right',"}}
    {{"    collapse_class: 'fa fa-caret-down',"}}
    {{"    edit_class: 'fa fa-pencil',"}}
    {{"    delete_class: 'fa fa-trash',"}}
    {{"    save_class: 'fa fa-save',"}}
    {{"    cancel_class: 'fa fa-remove',"}}
    {{"}"}}

Actions

Below Configuration needs to be enabled.

If Parent of records needs to be changed while editing then set edit_parent to true. 

    {{"actions: {"}}
    {{"    add: true,"}}
    {{"    edit: true,"}}
    {{"    delete: true,"}}
    {{"    edit_parent: true,"}}
    {{"}"}}

Column Editable

By default all columns are editable if edit set to true at the Grid level. However it can be managed at the column level through editable.

    {{"editable: true"}}
