Conditional Row Edit, Delete and CSS Class

If some rows in the grid shouldn't be edited or deleted then row_edit_function and row_delete_function methods can be set at the grid level. These methods will get the respected record(row data) as an argument.

Row Edit Function

    {{"row_edit_function: function(rec) {"}}
    {{"  if (rec.parent === 0) {"}}
    {{"    return false;"}}
    {{"  } else {"}}
    {{"    return true;"}}
    {{"  }"}}
    {{"}"}}

Row Delete Function

    {{"row_delete_function: function(rec) {"}}
    {{"  if (rec.parent === 0) {"}}
    {{"    return false;"}}
    {{"  } else {"}}
    {{"    return true;"}}
    {{"  }"}}
    {{"}"}}

If some rows need to be highlighted in the grid then row_class_function function can be used.

Row Class Function

    {{"row_class_function: function(rec) {"}}
    {{"  return 'row-custom';"}}
    {{"}"}}
