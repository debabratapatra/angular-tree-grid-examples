Filter Component

Config

Enable filter at grid level to enable filter toolbar. Filter can be disabled at column level. 

    {{"configs: any = {"}}
    {{"    filter: true,"}}
    {{"}"}}

Config

A custom filter function can be assigned to filter_function. 

    {{"filter_function: function(row_data, column, column_value, filter_value) {"}}
    {{"    return column_value > filter_value;"}}
    {{"}"}}

See weight column below. It gets Row data, column configuration, column value and filter value as arguments.
